Set your API keys in [settings](/settings), then use the API to generate CSV reports and manage your projects programmatically.

For manual data export, you can download csv files in your (calendar)[./calendar] view.

For integration with AI agents, use our MCP server at mcp.donethat.ai. Set it up e.g., in Claude under Customize > Add manual.
