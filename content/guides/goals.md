You can set goals for each week, month, quarter, or year after finishing one of these periods.

Goals can be set as input goals or output goals.

Input goals are on project- or -category level. You set how much time you want to spend on a specific category or project and DoneThat will do the accounting.

Output goals are things you want to achive. Donethat only logs those and you can check them off yourself. These can be again daily goals or e.g., quarterly OKRs.
